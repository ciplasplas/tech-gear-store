# Tech.no - Cửa hàng công nghệ

## Giới thiệu
Tech.no là website cửa hàng bán lẻ sản phẩm công nghệ được xây dựng dưới dạng đồ án cuối kỳ lập trình web. Dự án mô phỏng trải nghiệm mua sắm trực tuyến với giao diện bán hàng, trang sản phẩm, giỏ hàng và khu vực quản trị dành cho admin.

## Tính năng chính
- Trang chủ hiển thị banner, sản phẩm nổi bật và bộ lọc theo thương hiệu.
- Các trang danh mục: Laptop, PC Gaming, Phụ kiện.
- Trang chi tiết sản phẩm với thông tin, giá và nút mua ngay.
- Giỏ hàng để người dùng kiểm tra sản phẩm đã chọn.
- Trang đăng nhập đơn giản cho người dùng.
- Khu vực admin gồm trang tổng quan, quản lý sản phẩm, quản lý đơn hàng và hồ sơ admin.

## Cấu trúc dự án
- `index.html` - Trang chủ chính.
- `assets/css/` - Style cho giao diện website.
- `assets/images/` - Hình ảnh logo, banner và sản phẩm.
- `pages/` - Các trang phụ trợ như đăng nhập, giỏ hàng, admin và danh mục sản phẩm.

## Công nghệ sử dụng
- HTML
- CSS
- Font Awesome

## Triển khai
Dự án đã được host trên Vercel và có thể truy cập trực tiếp tại:

https://tech-gear-store.vercel.app

## Hướng dẫn truy cập
1. Mở trình duyệt.
2. Truy cập `https://tech-gear-store.vercel.app`.
3. Dùng menu điều hướng để xem trang chủ, sản phẩm, giỏ hàng và khu vực admin.

## Lưu ý
- Đây là một website tĩnh, chưa tích hợp backend hoặc cơ sở dữ liệu.
- Phù hợp để trình diễn thiết kế và trải nghiệm giao diện cửa hàng online.
